NS2 Installation Guide

Follow these steps to install NS2 on your Ubuntu system.

Step 1: System Requirements

Make sure your system meets the following requirements:

  • OS: Ubuntu 14.04 LTS (32-bit)
  • RAM: Minimum 2GB
  • Processor: 2.5 GHz and above

Step 2: Download NS2

Download the ns-allinone-2.35.tar.gz package from the following URL:

https://osdn.net/projects/sfnet_nsnam/downloads/allinone/ns-allinone-2.35/ns-allinone-2.35.tar.gz/

Step 3: Install Required Packages

Open a terminal and execute the following command:

sudo apt-get install gcc g++ python python-dev mercurial bzr gdb valgrind gsl-bin libgsl0-dev libgsl0ldbl flex bison tcpdump sqlite sqlite3 libsqlite3-dev libxml2 libxml2-dev libgtk2.0-0 libgtk2.0-dev uncrustify doxygen graphviz imagemagick texlive texlive-latex-extra texlive-generic-extra texlive-generic-recommended texinfo dia texlive texlive-latex-extra texlive-extra-utils texlive-generic-recommended texi2html python-pygraphviz python-kiwi python-pygoocanvas libgoocanvas-dev python-pygccxml

Step 4: Extract NS2

Extract the ns-allinone-2.35.tar.gz file to your home directory:

tar -xvf ns-allinone-2.35.tar.gz

Step 5: Change Directory

Change to the ns-allinone-2.35 directory:

cd ns-allinone-2.35

Step 6: Execute Install Script

Execute the install script:

sudo ./install

Step 7: Install NAM

Install the NAM package:

sudo apt-get install nam

Step 8: Install XGraph

Install the XGraph package:

sudo apt-get install xgraph

Step 9: Install FFmpeg

Install the FFmpeg package:

sudo apt-get install ffmpeg

Step 10: Verify Installation

Verify that NS2 has been installed successfully:

ns

Step 11: Run NS2

Run NS2:

ns your_script.tcl

Step 12: Visualize Results

Visualize the results using NAM:

nam your_script.nam

Congratulations! You have successfully installed NS2 on your Ubuntu system.
